# hsimply

Haskell implementation to read JSON files generated by the Python [`simply`](https://github.com/yourname/pysimply) package.

## Overview

`simply` is a simplified Python AST representation that can be serialized to JSON. `hsimply` parses that JSON into native Haskell AST data structures.

## Project Structure

- `src/Simply/AST.hs` — Haskell data types mirroring the Python `simply` AST.
- `src/Simply/JSON.hs` — `aeson`-based `FromJSON` / `ToJSON` instances for the tagged-union JSON format produced by `simply`.
- `app/Main.hs` — Small CLI executable that reads a JSON file and prints the parsed Haskell AST.
- `test/Main.hs` — Round-trip tests for expressions, statements, and modules.

## Building

```bash
cd hsimply
cabal build
cabal test
```

## Usage

### From Python

```python
from simply.parse_stmt import parse
import ast, json

code = "def foo(x): return x + 1"
tree = ast.parse(code)
stmts = [parse(stmt) for stmt in tree.body]
with open("ast.json", "w") as f:
    json.dump([stmt.json for stmt in stmts], f)
```

### From Haskell

```haskell
import qualified Data.Aeson as A
import qualified Data.ByteString.Lazy as BSL
import Simply.AST
import Simply.JSON ()

main = do
  bs <- BSL.readFile "ast.json"
  let modAst = A.eitherDecode bs :: Either String Module
  print modAst
```

Or use the provided executable:

```bash
./hsimply-exe ast.json
```

## Supported Constructs

All expression and statement nodes that are implemented in `simply` are supported, including:

- **Expressions**: `Name`, `Constant`, `BoolOp`, `BinOp`, `UnaryOp`, `Lambda`, `IfExp`, `Call`, `Attribute`, `Compare`, etc.
- **Statements**: `FunctionDef`, `Return`, `Delete`, `Assign`, `For`, `While`, `If`, `Raise`, `Assert`, `Import`, `ImportFrom`, `Expr`, `Pass`, `Break`, `Continue`, etc.
- **Auxiliary types**: `Identifier`, `Arguments`, `Arg`, `Keyword`, `Alias`, `ExprContext`, `Operator`, `UnaryOp`, `CmpOp`, etc.
